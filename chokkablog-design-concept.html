Design notes

This is the refined concept combining the minimal Swiss baseline (concept 3) with subtle bordered cards from concept 2.

  • Wordmark: "chokka." — bold sans-serif with coral dot accent
  • Accent colour: coral #D85A30 — used for labels, tags, tool links, and the dot
  • Post cards: light 1px #e8e8e8 border, 8px radius, hero image area + title/excerpt/meta
  • Tool cards: matching border treatment, name/description/launch link
  • Typography: system sans-serif, bold 700 for titles, clean hierarchy
  • Layout: 3-column grid on desktop, single column on mobile
  • Navigation: Writing / Tools / Archive / About
  • Hero images: placeholder chart sketches — in production these would be exhibit screenshots
  • Tools section: separated by a strong black rule, coral "TOOLS" label
  • Legacy archive: separate "Archive" nav item for migrated Blogger content
